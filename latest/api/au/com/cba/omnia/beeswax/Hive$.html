  43. def repair(database: String, table: String): Hive[Unit]

    Repair/Register partitions with the Hive table.

    Repair/Register partitions with the Hive table.

    Please be aware that if you have a large number of partitions, this could fail due to any of the following reasons :-

    • Out of memory on the client side.
    • Slow network would cause transportation issues while sending the huge object graph to the metastore.
    • Out of memory on the metastore side.

    In these situations, it would be best to use addPartitions.

    database

    Name of the database.

    table

    Name of table to create.
